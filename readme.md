# NotionPet 浏览器插件项目

浏览器插件项目

设计稿地址：https://www.figma.com/file/tV6sJHKc9C1MEFtG6ZCrL5/WidgetStore?node-id=24%3A14077

## 开始

```bash
# 项目启动
$ pnpm i
$ npm run dev
# 浏览器访问 chrome://extensions/
# 然后点击加载已解压拓展程序
# 选择项目下 dist 目录，即可安装插件进入插件开发模式

$ npm run dev-watch
# 此开发模式下会监听代码变化后自动重新构建
```
